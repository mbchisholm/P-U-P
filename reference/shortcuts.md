# Keyboard Shortcuts Reference

## Terminal (all platforms)

| Shortcut | What it does |
|---|---|
| `Tab` | Complete command, path, or argument |
| `↑ / ↓` | Navigate command history |
| `ctrl+c` | Kill the running process |
| `ctrl+l` | Clear the screen (history stays) |
| `ctrl+r` | Search history (type to filter) |
| `ctrl+a` | Jump to start of line |
| `ctrl+e` | Jump to end of line |
| `ctrl+k` | Delete from cursor to end of line |
| `ctrl+u` | Delete from cursor to start of line |
| `ctrl+w` | Delete previous word |
| `ctrl+d` | Exit current shell (EOF) |

## macOS Terminal.app

| Shortcut | What it does |
|---|---|
| `cmd+t` | New tab |
| `cmd+w` | Close tab |
| `cmd+option+←/→` | Switch tabs |
| `cmd+k` | Clear scrollback |
| `cmd+shift+{` / `}` | Switch tabs (alternate) |

## VS Code

| Shortcut | What it does |
|---|---|
| `cmd+p` / `ctrl+p` | Open file by name |
| `cmd+shift+p` / `ctrl+shift+p` | Command Palette |
| `cmd+,` / `ctrl+,` | Settings |
| `cmd+\`` / `` ctrl+` `` | Toggle integrated terminal |
| `cmd+shift+e` / `ctrl+shift+e` | File explorer |
| `cmd+shift+g` / `ctrl+shift+g` | Source control (git) |
| `cmd+shift+x` / `ctrl+shift+x` | Extensions |
| `cmd+b` / `ctrl+b` | Toggle sidebar |
| `cmd+/` / `ctrl+/` | Toggle comment |
| `option+↑/↓` / `alt+↑/↓` | Move line up/down |
| `cmd+d` / `ctrl+d` | Select next occurrence |
